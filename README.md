# prafiles

Building software at the intersection of infrastructure, data, AI, and real-world systems.

I work across backend engineering, cloud platforms, telemetry-heavy systems, and applied AI. A lot of my time goes into turning messy operational environments into reliable software products — especially where devices, data, automation, and human workflows all need to meet cleanly.

- Applied AI products, LLM integrations, and agentic workflows with guardrails
- Software for intelligent infrastructure, automation, and control-heavy environments
- Internal tooling, platform foundations, and developer productivity

## How I like to build

- Start from the operational reality, not the slide deck
- Prefer robust systems over clever abstractions
- Design for maintainability early
- Treat data models and interfaces as product decisions
- Use AI where it compounds leverage, not where it adds theater
- Push toward repeatability, visibility, and simpler deployments
